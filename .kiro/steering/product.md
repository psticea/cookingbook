# Product Overview

A bilingual (Romanian/English) recipe website for personal cooking recipes. The site features 50+ recipes organized by category, with filtering, search, and responsive design for both mobile and desktop.

## Key Features

- Bilingual support (Romanian default, English available)
- Recipe categories: breakfast, pasta, stir-fries, soups-and-stews, main-courses, burgers-and-wraps, salads-and-bites, basics
- Keyword-based filtering system
- Adjustable text size (3 levels)
- Dark/light theme support
- Recipe scaling (servings adjustment)
- Personal notes section for each recipe
- Cooking basics page with fundamental techniques
- About page
- Responsive design (mobile-first)
- Recipe images: 1200x1200px squares
- Hosted on Azure Static Web Apps

## User Preferences

All user preferences (language, theme, text size) are persisted in localStorage and maintained across sessions.
